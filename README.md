**EE-Paper-Agent-Workflow**
An automated multi-agent workflow designed for Electronic Engineering research and postgraduate preparation. 

**Features:**
* **Parser Node**: Extracts complex layouts and raw data from advanced semiconductor physics papers (e.g., IEEE Transactions).
* **Reviewer Node**: Leverages large context window models to derive underlying physical models (e.g., MOSFET characteristics, Defect Formation Energy).
* **Tutor Node**: Automatically generates targeted exam questions for Subject 821 based on extracted knowledge.

*Currently processing 10+ papers daily with high token throughput.*
